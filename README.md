# Mini University System

A web-based mini university management system built using ASP.NET Core MVC and C#. The system is designed to simulate core academic operations such as student registration, course management, exams, and grading.

---

## 🚀 Features

- Student management (create, update, delete, view)
- Lecturer management
- Course management
- Department and class structure
- Course registration (many-to-many relationship)
- Exam and assessment tracking
- Student grading system
- Dashboard with summary statistics

---

## 🧠 Key Concepts Implemented

- ASP.NET Core MVC architecture
- Entity Framework Core (Code First)
- One-to-many and many-to-many relationships
- LINQ queries for data retrieval
- ViewModels for clean data presentation
- Separation of Concerns (Controllers, Models, Views)

---

## 🛠️ Tech Stack

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap (UI)

---

## 📊 Project Purpose

This project was built to practice real-world backend development concepts including relational database design, CRUD operations, and system structuring in a university-like environment.
